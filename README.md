# todo-app

## Frontend setup
```
cd todo-app
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

## API setup
```
cd api
npm install
```

### Serve api
```
node index.js
```
